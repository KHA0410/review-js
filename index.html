⭐ Number

isNaN() để kiểm tra đối tượng không phải là số

.toFixed() để làm tròn Number, nhận tham số truyền vào là số thập phân muốn làm gọn tới

⭐ String

+ Sử dụng backslash \ , `${chuỗi}`
+ Dùng .length để kiểm tra độ dài chuỗi
+ myString.indexOf("string tìm") tìm vị trí trái qua, có thể truyền vào index bắt đầu tìm
+ myString.lastIndexOf() tìm vị trí từ phải qua trái
+ myString.slice(index start, index end) cắt chuỗi
+ myString.replace("chuỗi cần tìm", "chuỗi cần thay thế") tìm và thay thế trong chuỗi, /chuỗi cần tìm/g đây là biểu thức chính quy tìm tất cả phần tử giống chuỗi cần tìm và thay thế
+ myString.toUpperCase() chuyển tất cả chữ trong chuỗi thành chữ hoa
+ myString.toLowerCase() chuyển tất cả chữ trong chuỗi thành chữ thường
+ myString.trim() xóa bỏ khoảng các đầu cuối của chuỗi
+ myString.split() cắt string thành array dựa vào điểm chung, nếu truyền chuỗi rỗng sẽ cắt từng chữ cái thành array

⭐ Toán tử số học(Arithmetic)

+, -, *, **, /, %, ++, --

⭐ Toán tử gán(Assignment)

=, +=, -=, *=, /=, **=

⭐ Toán tử ++ và --

Prefix(++a, --a) & Postfix(a++, a--)
* Prefix: Thực hiện phép tính tiền tố(+1, -1) trước -> rồi mới lấy giá trị a làm các hành động sau.
* Postfix: Lấy giá trị ban đầu của a thực hiện các hành động trước -> rồi mới thực hiện phép tính hậu tố(+1, -1).

Ex: var a = 6;
var b = a++ + --a;
-> b = 6 + 6 = 12

⭐ Toán tử so sánh(Comparison)

==, !=, >, <, >=, <= Chỉ so sánh value(1 == '1' => true)
===, !== So sánh cả value và datatype(1 === '1' => false)

⭐ Toán tử logic(Logical)

&&, ||, !

&&: var result = 'a' && 'b' && 'null' && 'c' => result = null (đọc từ trái qua nếu phát hiện falsy thì return)
||: var result = 'a' && 0 && 'null' && 'c' => result = a (đọc từ trái qua nếu phát hiện truthy thì return)

⭐ Hàm (Function)

Có 2 loại là hàm built-in(được js định nghĩa sẳn) và hàm tự định nghĩa
Note:
- Phải gọi thì hàm mới thực thi
- Có thể nhận tham số
- Có thể trả về 1 giá trị
- Tham số và đối số có thể là bất kể kiểu dữ liệu nào
- Tham số của hàm có tính private nên chỉ dùng được bên trong hàm đó
- Nếu truyền thiếu đối số hàm vẫn chạy nhưng tham số bị thiếu sẽ là undefined

Structure:
function sum(tham số 1, 2,...){
Các câu lệnh cần thực thi
}
- Gọi hàm để thực thi các câu lệnh
sum(đối số1, 2, ...)

Arguments
Tại sao các hàm như console.log() lại có thể truyền vô số đối số, chẳng lẽ js định nghĩa sẳn ư ?
Ex:
function writeLog(){
console.log(arguments) => ['log1', 'log2', 'log3']
}
writeLog('log1', 'log2', 'log3')

Return
Hàm không dùng return, kết quả sai sẽ cho ra undefined

⭐ Các loại hàm

- Declaration function (có cơ chế hosting)
function countNumber(){}
- Expresstion function
var countNumber = function(){}
- Arrow function
var countNumber = () => {}

Note: Cơ chế hosting là gọi hàm trc cả khi nó được định nghĩa

⭐ Mảng(Array)

var languages = ["JS", "C++", "C#", 123, undefined, function(){}, null, ...];

- Nội dung trong [] gọi là các "phần tử" của mảng languages
- Các phần tử của mảng có thể là bất kì kiểu dữ liệu nào
Các phương thức hay dùng với array:
1. Array.isArray(languages) để kiểm tra biến có phải array không
2. languages.length để kiểm tra mảng có bao nhiêu phần tử
3. languages.[n] lấy ra phần tử thứ n
4. languages.toString chuyển Array thành String
5. languages.join('--') chuyển Array thành String và ngăn cách các phần tử bằng --
6. languages.pop() xóa phần tử cuối mảng và trả ra phần tử đó, khi hết phần tử thì trả về undefined
7. languages.shift() xóa phần tử đầu mảng và trả ra phần tử đó, khi hết phần tử thì trả về undefined
8. languages.unshift() thêm 1 hoặc nhiều phần tử vào đầu mảng
9. languages.push() thêm 1 hoặc nhiều phần tử vào cuối mảng, tăng độ dài mảng
10. languages.splice(index, số phần tử muốn xóa, phần tử muốn chèn) dùng để xóa(input 2 tham số), chèn(input 3 tham số) phần tử bất kì nếu đã xác định được index
11. languages.concat(languages2) nối mảng languages2 vào languages
12. languages.slice(index, số phần tử muốn cắt) cắt 1 or nhiều phần tử, có thể cắt từ cuối lên bằng cách truyền số âm. Ngoài ra có thể truyền 0 để copy toàn bộ các phần tử của mảng
